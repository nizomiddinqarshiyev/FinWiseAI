# FinHealth-AI (demo)

Repo tuzilmasi va minimal boshlang'ich fayllar.

- frontend: React demo
- ml_models: ML stublar

Ishga tushirish:
1. Python virtualenv yarating, backend/requirements.txt o'rnating.
2. backend/api.py ishga tushiring: python api.py
3. frontend uchun npm install va npm start (agar kerak bo'lsa).
